Campfire design studio

Your workshop. Every graphic and UI piece in the game, in one place. Edit theme.css (colors, fonts, layout) or art.js (the drawings), save, and refresh this page — no need to play a game to see your changes.

The campfire

Drawn in campfireSVG() in art.js. Drag the slider to preview the fire at any point in the game — it burns down as the rounds go by.

The smoke buffooneries

These drift up through the smoke on the home screen. They live in the VIGNETTES array in art.js — each one is a small standalone SVG. Edit one, or copy the pattern to add your own (it joins the rotation automatically).

The marshmallows

Drawn by mallow(type, size) in art.js. Golden = best-story vote, burnt = worst-story vote.

Design tokens

The variables at the top of theme.css. Change a value there and everything using it updates.

UI components

The building blocks of every menu, styled by theme.css. What you see here is exactly what the game screens use.
